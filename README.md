# TETRIS using THREEJS

```bash
$ npm install
$ npm run dev
```

## Playing Game
- press <kbd>P</kbd> to start
- navigate minos using corresponding <kbd>arrowkeys</kbd>(except for UP)
- press <kbd>R</kbd> to rotate clockwise, <kbd>E</kbd> to rotate counterclockwise
